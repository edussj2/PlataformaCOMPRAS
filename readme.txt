•	Core: Configuración General del sistema
    o	configBaseDatos: Configuración de constantes de la base de datos.
    o	configGeneral: Configuración de constantes de la empresa (Dominio y nombre)
    o	mainModel: Modelo principal, funciones de conexión con la base de datos y otras funciones generales.
•	Ajax: Carpeta que contiene los archivos Ajax que invocaran a los controladores.
•	Controllers: Es la carpeta que contiene los archivos controladores, es donde preparamos los datos para enviar a la base de datos a través de los modelos.
•	Files: Contiene todas las imágenes de los productos que sean agregados.
•	Models: Es la carpeta que contienes los archivos modelo, donde están los métodos de CRUD de cada tabla.
•	Views: Contiene todas las vistas de cada uno de los módulos
    o	js: Contiene los scripts js
    o	Libreries: Dependencias agregadas como Bootstrap jQuery, swwetAlert2 y select2.
    o	css: Contiene los scripts para dar estilos.
    o	img: Contiene imágenes como el logo - avatar (Imágenes de avatares).
    o	Components: Contienes en archivos php las partes de la vista que se repiten constantemente.
    o	Content: Todos los formularios y vistas de la plataforma.
